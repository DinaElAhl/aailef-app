# AAILEF — إطار تقييم المخرجات اللغوية العربية للذكاء الاصطناعي

> أداة ويب تقيّم النصوص العربية المُولّدة بالذكاء الاصطناعي عبر **عشرة محاور لغوية موزونة** باستخدام Claude.
> *(English README: [README.md](./README.md))*

تنتج الأداة درجة جودة إجمالية، ومخطط رادار، وتحليلاً لكل محور مع سببه، وأنواع الأخطاء المكتشفة، وتوصيات عملية للتحسين. صُمّمت لدعم تقييم منهجي قابل للتكرار لمخرجات نماذج اللغة العربية — للأبحاث والرسائل العلمية وضبط الجودة.

الأداة من نوع **«أحضري مفتاحك»**: كل مستخدِم يُدخل مفتاح Anthropic API الخاص به، ويُحفظ المفتاح **في المتصفح فقط** ويُرسَل **مباشرةً** إلى Anthropic. لا يوجد خادم خلفي ولا مفتاح مشترك، لذا يمكن نشر الأداة مجاناً على GitHub Pages واستخدامها من أي مكان في العالم.

---

## ✨ الميزات

- **مرجعية تقييم من عشرة محاور موزونة** (A1–A10) مع فصل بين جودة اللغة (LQ) وجودة المهمة (TQ).
- **مخرجات مُقيَّدة بمخطط (Structured Outputs)** — يُعيد النموذج JSON صحيحاً في كل مرة، فلا تفشل قراءة النتائج أبداً.
- **نموذج Claude Opus 4.8** للأحكام اللغوية العربية الأدقّ، مع خيار **تفكير معمّق** اختياري.
- **مخطط رادار + أشرطة لكل محور + سبب + تصنيف الأخطاء.**
- **سجل التقييمات** (آخر 50، محفوظ في المتصفح) مع **عرض للمقارنة** بين الأنظمة.
- **تصدير** نتيجة واحدة بصيغة JSON أو كامل السجل بصيغة CSV (UTF-8 مع BOM، يفتح بسلاسة في Excel) — جاهز لجداول بيانات الرسالة.
- واجهة عربية كاملة من اليمين إلى اليسار.

## 🧮 الإطار

| المحور | الاسم | الإنجليزية | الوزن |
|:------:|------|------------|:-----:|
| A1 | سلامة الكتابة | Orthographic Integrity | ×1.0 |
| A2 | التشكيل والغموض | Diacritics & Ambiguity | ×1.5 |
| A3 | الاتساق الصرفي | Morphological Consistency | ×1.5 |
| A4 | المطابقة النحوية | Grammatical Agreement | ×1.0 |
| A5 | سلامة التركيب | Syntactic Well-formedness | ×1.0 |
| A6 | وضوح المعنى | Semantic Clarity | ×1.5 |
| A7 | الحساسية للسياق | Context Sensitivity | ×2.0 |
| A8 | ثبات السجل اللغوي | Register & Dialect Control | ×1.5 |
| A9 | دقة المصطلح | Terminological Accuracy | ×2.0 |
| A10 | جودة أداء المهمة | Task Performance Quality | ×2.0 |

**طريقة الحساب:**

- `LQ` (جودة اللغة) = المتوسط المُرجَّح لمحاور **A1–A9** على مقياس 1–5.
- `TQ` (جودة المهمة) = درجة المحور **A10** مباشرةً.
- **النتيجة الإجمالية** = `0.7 × LQ + 0.3 × TQ`.

يُمنح كل محور درجة من 1 إلى 5 مع سبب موجز مستند إلى دليل، ورمز خطأ (E1–E9 أو «لا يوجد»). التعريفات الكاملة في الملف [`src/aailef.js`](./src/aailef.js).

## 🚀 التشغيل محلياً

يتطلّب [Node.js](https://nodejs.org) الإصدار 18 أو أحدث.

```bash
npm install
npm run dev
```

افتحي الرابط الظاهر (افتراضياً <http://localhost:5173>)، ثم من تبويب **⚙️ الإعدادات** الصقي مفتاح Anthropic، وابدئي التقييم.

> احصلي على مفتاح من <https://console.anthropic.com/settings/keys>.

لبناء نسخة الإنتاج:

```bash
npm run build      # الناتج في dist/
npm run preview    # معاينة النسخة المبنية محلياً
```

## 🌍 النشر مجاناً على GitHub Pages

يُنشَر هذا المستودع تلقائياً على GitHub Pages عند كل دفع إلى `main` عبر ملف العمل
المُضمَّن ([`.github/workflows/deploy.yml`](./.github/workflows/deploy.yml)): يبني
التطبيق (`npm run build`) وينشر مجلد `dist/`. الموقع متاح على
`https://<username>.github.io/<repo>/`.

لإعداد ذلك على نسخة جديدة (fork):

1. ادفعي المشروع إلى فرع `main`.
2. في المستودع: **Settings → Pages → Source: GitHub Actions**.
3. عندها يُعاد البناء والنشر تلقائياً عند كل دفع إلى `main`.

يستخدم البناء مساراً نسبياً (`base: "./"`)، لذا يعمل كما هو أيضاً على Netlify وVercel وCloudflare Pages وأي استضافة ثابتة دون إعداد.

## 🔐 الخصوصية والأمان

- يُحفظ مفتاحك عبر `localStorage` **في متصفحك أنت**، ولا يُرسَل لأي جهة سوى `api.anthropic.com`.
- يتصل التطبيق بواجهة Anthropic مباشرةً من المتصفح. هذا آمن **لأن كل مستخدِم يستعمل مفتاحه الخاص** — لا يوجد مفتاح خادم مشترك قد يتسرّب.
- تُرسَل النصوص المُقيَّمة إلى Anthropic للتقييم (كأي استدعاء لـ Claude). لا تلصقي محتوى لا يُسمح لكِ بمشاركته مع طرف ثالث.
- لا تتبّع ولا تحليلات ولا خادم خلفي.

## 🛠️ البنية التقنية

- **React 18 + Vite** (تطبيق صفحة واحدة، بلا خادم خلفي).
- `src/aailef.js` — بيانات الإطار ومعادلات الحساب (المصدر الموثوق).
- `src/api.js` — عميل Claude: بناء الموجّه، والمخرجات المُقيَّدة بمخطط، ومعالجة الأخطاء.
- `src/App.jsx` — الواجهة (تبويبات: التقييم / السجل / الدليل / الإعدادات).

النموذج: `claude-opus-4-8`. لتغييره، عدّلي `MODEL` في `src/aailef.js`.
